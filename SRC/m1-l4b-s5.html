Lesson 1.4b

Apply Step 8: Audit the Scope Trace

Purpose

Conduct a forward scope trace on six MRHN decomposition components. Check whether each component is assigned to the correct parent SI. Find the one misattribution and correct it.

Instructions

  1. Review each component's listed parent SI. Judge whether the attribution is correct.
  2. If a component is misattributed, flag it and select the correct parent SI from the dropdown.
  3. Select Check My Work to see expert reasoning for all six components.

Success Criteria

  • The one misattributed component is correctly identified
  • The correct parent SI is selected and the reason is clear
  • The five correctly attributed components are not flagged as errors

Expected Evidence

Six scope trace judgments with one corrected parent attribution. Expert reasoning is shown for all six components after you submit, regardless of your answers.

Shift your focus. Screen 4 asked about granularity: can each component be taught and assessed as one unit? This screen asks about attribution: is each component assigned to the correct parent SI? The audit logic is different. You are checking the map's structure, not the size of its pieces.
Quick Reference: Constraint Codes

RC-1 (HIPAA 45 CFR 164.308(a)(5)): Workforce security training is a covered entity obligation. The course must fulfill it documentably -- the training must exist, be delivered, and be verifiable for audit purposes.

RC-2 (IR-2024 Alignment): Course content must align to IR-2024 as adopted, not a revised or idealized version. Scope items and decomposition components draw from the protocol in its current adopted form.

RC-3 (OCR 18-Month Timeline): Delivery must occur within the OCR audit remediation window. The timeline is a binding constraint -- it governs scheduling regardless of project or instructional preferences.
